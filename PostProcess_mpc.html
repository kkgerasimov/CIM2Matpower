PostProcess_mpc module¶

The PYPOWER extension converts Case objects to PYPOWER cases and writes results from a power flow calculation back to the orignal case instance. The module PostProcess_CIM_files defines function to be used for preprocessing CIM files before they are parsed with the PyCIM RDFXMLReader. Most often it means fixing certain HTML tags.

Note

This module is based mostly on the the started but never finished OpenSource project Cim2BusBranch https://www.versioneye.com/Python/Cim2BusBranch/0.1
